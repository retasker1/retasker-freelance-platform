# Пример баннера

Этот файл можно удалить после добавления реального баннера.

## Для тестирования:

1. Создайте изображение 1200x700 пикселей
2. Сохраните как WebP, JPG или PNG
3. Переименуйте в "banner.webp" (или другое имя)
4. Обновите app/config/banner.ts

## Текущая конфигурация:
- Файл: banner.webp
- Ссылка: https://example.com
- Статус: включен

Если файл banner.webp не найден, будет показан градиентный баннер.
